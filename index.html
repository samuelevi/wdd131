WDD 131 – Dynamic Web Fundamentals

This course introduces the fundamentals of web development using HTML, CSS, and JavaScript. Students will learn to create responsive, accessible, and user-friendly web pages. Throughout the course, you'll build projects that demonstrate your understanding of web design principles, semantic HTML, styling techniques, and dynamic JavaScript functionality.

Learning Activities

Throughout this course, you will complete various learning activities including:

  • HTML structure and semantic markup
  • CSS styling, layout, and responsive design
  • JavaScript fundamentals and DOM manipulation
  • Web accessibility and best practices
  • Performance optimization and testing
These activities build upon each other to create a comprehensive understanding of modern web development.
